The name gives it up, analysis! Exploratory Data Analysis is the process of analyzing given data and drawing meaningful insights from it. Technically, the process which helps to :

  1. uncover underlying structure in data.
  2. uncover the relations among attributes.
  3. uncover the dependencies among attributes.
  4. uncover the important attributes concerning the dependent variable.
  5. provide with insights that might be helpful before applying any model
